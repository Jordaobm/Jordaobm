### Sobre mim

Sou desenvolvedor de software há cinco anos, todos na mesma empresa — a Supera Inovação em Tecnologia — onde comecei como Desenvolvedor Júnior I e hoje atuo como **Líder Técnico**. Essa progressão interna é, na prática, meu principal case: fui de executar tarefas simples a ser a referência arquitetural de um time, decidindo stack, padrões de projeto e estrutura de sistemas novos.

Hoje conduzo três projetos de abrangência nacional para a administração pública, incluindo dois rebuilds simultâneos de sistemas estaduais — modernizando aplicações Groovy/Grails para React, TypeScript e Java Spring Boot. Também estruturei o processo de definição de contratos REST entre frontend e backend antes de cada sprint, o que reduziu retrabalho e tornou as entregas mais previsíveis para o time inteiro.

No dia a dia, escrevo um pouco menos código do que antes: coordeno cerimônias ágeis, converso direto com clientes sobre prazos e homologações, reviso código, mantenho pipelines de CI/CD e integro novos desenvolvedores ao time.

Trajetória completa (Júnior I → Líder Técnico), formação e certificações: [jordao.dev/sobre](https://jordao.dev/sobre) · [currículo em PDF](https://jordao.dev)

Um repositório para guardar, testar e mostrar o que aprendi como desenvolvedor — cada rota é uma demonstração isolada e funcional de uma capacidade real: autenticação (JWT, OAuth, TOTP, WebAuthn/Passkeys), PWA instalável, 3D interativo, captura de mídia, push notifications, SQLite rodando via WebAssembly num Web Worker, e mais. Hoje são **9 demos funcionando de ponta a ponta**, com i18n em três idiomas, acessibilidade WCAG 2.1/2.2 AA e testes E2E via Playwright cobrindo 90,1% do código.

O projeto recebe o mesmo cuidado de engenharia (Feature-Sliced Design, contratos tipados, tratamento de erro, testes) que aplico nos sistemas em produção que conduzo no meu trabalho atual.
